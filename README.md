# Payroll Management System

This repository contains a full-stack Payroll Management System built with a Node.js backend and a React frontend.

## Project Overview

- **Backend**: Express.js server with MongoDB and Mongoose.
- **Frontend**: React application for admin and employee dashboards.
- **Features**:
  - Employee attendance tracking
  - Salary management and editing
  - Leave request handling
  - Loan request handling
  - Provident fund management
  - User profiles and authentication

## Repository Structure

- `backend/` - API server, database models, and backend logic.
- `frontend/` - React web application and UI components.

## Setup Instructions

1. Install dependencies for the backend:
   ```bash
   cd backend
   npm install
   npm start
   ```

2. Install dependencies for the frontend:
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. Open the React app in your browser when the frontend server starts.

## Notes

- The backend uses `nodemon` for development.
- The frontend uses `react-scripts`.
- This repository now uses the `main` branch for the updated project code.
